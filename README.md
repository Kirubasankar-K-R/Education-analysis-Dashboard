# Education-Analysis-Dashboard
# 🎓 School Performance Tracker Dashboard | Power BI Project

## 1. Project Overview
The School Performance Tracker Dashboard is an interactive Power BI project developed to analyze and monitor student academic performance across Secondary School, 11th Grade, and 12th Grade levels.  

This dashboard provides comprehensive insights into student attendance, academic scores, ranking analysis, teacher ratings, gender distribution, and section-wise performance through dynamic and visually engaging reports.

---

## 2. Project Objective
The primary objectives of this project are:

1. To track student academic performance  
2. To analyze attendance and ranking trends  
3. To compare section-wise student results  
4. To monitor gender-based academic distribution  
5. To identify toppers and performance patterns  
6. To create an interactive educational analytics dashboard using Power BI
